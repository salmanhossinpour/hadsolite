## 📖 درباره بازی (About the Game)

**حدسولایت** یک بازی کلمات تعاملی، مینیمال و فوق‌العاده جذاب به زبان شیرین پارسی است که ایده گرفته از بازی محبوب آمیرزا است. در این بازی شما در نقش یک کاتب دانا به دیوان کلمات معلق وارد می‌شوید و با کشیدن خطوط بین حروف معلق، کلمات اصیل و پنهان فارسی را پیدا می‌کنید.

این بازی به سفارش و طراحی اختصاصی **سلمان حسین** و با اهداف زیر توسعه داده شده است:
* 🌟 سرگرمی اصیل و ارتقای معلومات واژگان فارسی.
* 👨‍👩‍👧‍👦 ایجاد فضایی گرم برای بازی‌های خانوادگی و دورهمی‌ها (خونوکی بازی!).
* 🎨 ظاهری چشم‌نواز، اصیل و مدرن با تم تاریک مینیمال "دیوان حدسولایت".

---

## ✨ ویژگی‌های برجسته (Key Features)

* **🧩 گیم‌پلی مهیج و روان کلمات:** اتصال روان حروف به صورت دایره‌ای با انیمیشن‌های ملایم و دلنشین.
* **📅 چالش روزانه حدسولایت:** هر روز یک چالش مجزا با تقویم سنتی دیوان برای کسب جوایز ویژه و محک زدن روزمره معلومات.
* **🏆 جدول رده‌بندی هوشمند (Leaderboard):** ردیابی آنلاین امتیازات شما و رقابت تنگاتنگ با سایر کاتبان دیوان و کاتبان افسانه‌ای!
* **🎒 دکان سکه حدسولایت (Shop):** خرید ایمن بسته‌های سکه با مکانیزم کارت‌به‌کارت مستقیم و تایید و فعال‌سازی فوری از طریق پشتیبانی اختصاصی تلگرام.
* **🛡️ ذخیره‌سازی امن و دیتابیس بومی (NeDB):** ذخیره پیشرفت، سکه‌ها و دستاوردهای شما به صورت لحظه‌ای و امن در دیتابیس NeDB در سمت سرور به همراه سیستم سنترالیزه جلوگیری از تقلب و ایمن‌سازی ورودی‌ها.
* **🎵 نوای سنتی سنور:** موسیقی ملایم و افکت‌های صوتی کلیک سنتی که حس دگرگونی و آرامش کاتبان قدیمی را القا می‌کند.

## 🛠️ ساختار فنی و امنیت (Tech Stack & Security)

این پروژه از یک معماری تمام‌پیوند (Full-stack) مدرن، سبک و با کارایی بالا بهره می‌برد:

* **Frontend:** 
  - **React 18** همراه با **Vite** جهت بارگذاری سریع و فاقد تاخیر.
  - **Tailwind CSS** برای پیاده‌سازی یک دیزاین کاملاً ریسپانسیو (موبایل و دسکتاپ) بر پایه سیستم رنگی مدرن `oklch`.
  - **Motion (Framer Motion)** جهت روان‌سازی حرکت واژگان و انیمیشن‌های لمس صفحه.
* **Backend:** 
  - **Express.js** به عنوان موتور مدیریت روترها و وب سرویس‌ها.
  - **NeDB (@seald-io/nedb):** یک دیتابیس درختی سبک و ذخیره اطلاعات در قالب فایل‌های سند به صورت ایمن و بهینه بدون نیاز به نصب موتورهای سنگین پایگاه داده.
* **Security & Anti-XSS:**
  - سیستم پاکسازی هوشمند و دقیق تمام ورودی‌های کاربر (مانند نام بازیکن در جدول رده‌بندی) برای مسدود کردن حملات تزریق کد (XSS).
  - استفاده از شناسه امنیتی یکتایِ تولید شده سمت سرور (UUID v4) برای ردگیری بازیکنان به صورت ناشناس و کاملاً محافظت شده.

## 🚀 راهنمای راه‌اندازی پروژه (Local Setup)

برای اجرای این بازی به صورت محلی بر روی سیستم خود، مراحل زیر را دنبال کنید:

### پیش‌نیازها
* نصب بودن **Node.js** (نسخه ۱۸ یا بالاتر)

### مراحل نصب و اجرا
۱. مخزن را کلون کنید:
   ```bash
   git clone https://github.com/your-username/hadsolite.git
   cd hadsolite
   ```

۲. وابستگی‌های پروژه را نصب کنید:
   ```bash
   npm install
   ```

۳. سرور توسعه همزمان کلاینت و سرور را اجرا کنید:
   ```bash
   npm run dev
   ```
   *بازی روی پورت `3000` به آدرس `http://localhost:3000` در دسترس خواهد بود.*

۴. برای ساخت نسخه تولیدی (Production Build):
   ```bash
   npm run build
   ```

۵. اجرای نسخه بیلد شده نهایی:
   ```bash
   npm start
   ```
